Potential Careers

The tech industry is constantly growing, and it's hungry for new employees. Jobs are well paid, and graduates are in demand here and overseas.

$88,000 - The median base salary for ICT professionals in Wellington. Absolute IT January 2016

Software Developer

Software developers develop and maintain computer software and websites.

Software developers with some experience usually earn:
$58K-$80K per year

Senior developers with several years' experience usually earn:
$80K-$110K per year

Security Analyst

Security analysts protect the information systems and computer networks of an organisation so information cannot be illegally accessed. They create security processes and monitor the IT security framework.




Security analysts earn an average of:
$80K-$140K per year

Electronics Engineer

Electronics engineers study, design and oversee production of electronic equipment such as radios, televisions, computers, washing machines and telecommunication systems. They may also service equipment.

Newly qualified electronics engineers usually earn:
$50K-$60K per year

Experienced electronics engineers usually earn:
$80K-$120K per

Web Developer

Web developers plan, produce and maintain websites using web programming languages, software applications, and databases.




Web developers with some experience usually earn:
$40K-$60K per year

Game Developer

Game developers write, design, program, animate, and test games and applications for computers, gaming consoles, and cellphones.

New game developers usually earn:
$49K-$60K per year

Game developers with experience usually earn:
$60K-$80K per year
